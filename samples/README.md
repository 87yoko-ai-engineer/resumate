# samples（テスト・デモ用のサンプル）

ResuMate の動作確認・デモ用の **架空データ** です（実在の人物・企業ではありません）。

## 中身
- **【サンプル求人・架空】DX推進部　管理業務AIスペシャリスト.txt** — 「① 求人情報を貼り付け」に貼って分析を試すための求人サンプル。
- **履歴書サンプル.pdf / 履歴書サンプル.html** — 架空の人物「山田 拓海」（経理 × Python・LLM API）の履歴書・職務経歴書。画像OCRの読み取りテストやスクリーンショットに使えます（HTMLはブラウザでそのまま開けます）。
- **履歴書サンプル（箇条書き）.md** — 上記の内容を箇条書きにしたもの。
- **sample-resume.json / フォームに読み込む方法.md** — 上記の内容を入力フォームに一括で読み込むためのデータと手順（ブラウザのコンソールに貼り付け）。

## 使い方の例（画像OCRを試す）
1. `履歴書サンプル.pdf`（または `履歴書サンプル.html`）を開く。
2. `Win + Shift + S` でスクリーンショットを撮る。
3. ResuMate の「② 履歴書作成 → 履歴書・職務経歴書を読み取り」で `Ctrl + V`。
4. 学歴・職歴・資格・職務経歴などが読み取られ、確認画面でチェックして反映できることを確認。

※ サンプルの個人情報はすべて架空です。実際の利用では、ご自身の個人情報が画像に写らないようご注意ください。
